# NBA pressure bar plot example data

A small example dataset used to demonstrate the
[`nba_pressure_bar_plot()`](https://sanbi-nba.github.io/nbaR/reference/nba_pressure_bar_plot.md)
function.

## Usage

``` r
NBA_example_press_bar_data
```

## Format

`NBA_example_press_bar_data` A data frame with 3 variables:

- Taxon:

  Taxonomic group (e.g., Birds, Mammals, Plants)

- pressure:

  Name of the pressure category

- percentage:

  Percentage value for each pressure
