# NBA combined ecosystem type protection level and threat level example data

This data is used to make a combined plot of protection level and threat
status of ecosystems within a functional group.

## Usage

``` r
NBA_example_comb_data
```

## Format

### `NBA_example_comb_data`

A data frame with 16 rows and 11 columns:

- OVERALL types:

  Ecosystem functional group name

- Well Protected,Moderately Protected,Poorly Protected,Not Protected:

  Protection levels

- Critically Endangered,Endangered,Vulnerable,Least Concern:

  Threat status

- TOT:

  Total number of ecosystems

- metric:

  Column to differentiate rows into groups
