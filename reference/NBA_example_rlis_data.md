# NBA index data (RLIs taxon)

This data is used to make a line plot of the red list index across a
number of years

## Usage

``` r
NBA_example_rlis_data
```

## Format

A data frame with 63 rows and 6 variables:

- Year:

  assessment year

- RLI:

  Red List Index value

- min:

  minimum value

- max:

  maximum value

- Assessment_Year:

  year of assessment

- Taxon:

  taxon name
