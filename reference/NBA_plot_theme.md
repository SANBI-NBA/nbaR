# NBA theme for a ggplot object

This function will create a theme for all ggplot graphs to align them to
the NBA "look". All NBA functions already use this theme.

## Usage

``` r
nba_plot_theme()
```

## Value

Returns a ggplot object with the NBA_theme styling
