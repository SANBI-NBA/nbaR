# NBA ecosystem type threat status data

This data is used to make a bar plot of threat status of ecosystems
within a functional group. This dataset has a totals column, but this is
not necessary.

## Usage

``` r
NBA_example_thr_data
```

## Format

### `NBA_example_thr_data`

A data frame with 8 rows and 6 columns:

- OVERALL types:

  Ecosystem functional group name

- Critically Endangered,Endangered,Vulnerable,Least Concern:

  Threat status

- TOT:

  Total number of ecosystems
