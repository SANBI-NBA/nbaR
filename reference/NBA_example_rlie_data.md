# NBA index data (RLIe)

This data is used to make a line plot of the red list index across a
number of years

## Usage

``` r
NBA_example_rlie_data
```

## Format

A data frame with 44 rows and 8 variables:

- Year:

  assessment year

- RLIE:

  Red List Index value

- lower:

  minimum value

- upper:

  maximum value

- criteria:

  Overall

- total_weight:

  weight

- total_count:

  count

- Biome:

  biome name
