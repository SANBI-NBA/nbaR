# NBA index data (EPLI 2024)

This data is used to make a line plot of the ecosystem protection level
index across a number of years

## Usage

``` r
NBA_example_epli2024_data
```

## Format

A data frame with 11 rows and 4 variables:

- X:

  row number

- T_BIOME:

  biome name

- Total:

  total number

- EPLI:

  EPLI
