# NBA map example data, mem (marine ecosystem map) 2023

This data is used to make a map of marine pelagic ecosystems and their
protection level

## Usage

``` r
NBA_example_map_data
```

## Format

### `NBA_example_map_data`

A data frame with 13 rows and 3 columns:

- P_EcosysType:

  Ecosystem name

- geometry:

  polygons

- protection_level:

  protection level

- threat_status:

  threat statis
